1. APP 管理
罗列出用户所创建的所有的 app

2. 创建 APP
APP名称： app 的名字
平台： iOS 安卓 自定义 三种
自定义下面： 只有一个功能，自定义事件。选择自定义，则不能设置崩溃，HTTP 还有自定义事件的开启和关闭。
是否开启崩溃上报：
  用户是否能够上报崩溃信息
  如果关闭，用户不能上报崩溃信息。页面这个功能则不能点击。
是否开启HTTP上报：
  用户是否能够上报 HTTP 信息
  如果关闭，用户不能上报 HTTP 信息。页面这个功能则不能点击。
是否开启自定义事件上报：
  用户是否能够上报自定义事件 如果关闭，
  用户不能上报崩溃信息。页面这个功能则不能点击。

3.功能选择
  选择某个功能

APP 崩溃日志：
  展示崩溃列表，点击某一行，查看崩溃的详细信息
网络诊断日志：
  展示网络诊断日志列表，点击某一行，查看网络诊断日志的详细信息
应用分析：
  请求分析：下载速度
  错误分析：HTTP 错误率、网络错误率
视频分析：
  CDN 竞品比较
  CDN 竞品比较 - 包含特定CDN
  CDN 历史总览
  CDN 实时总览
  CDN 主机分布总览
自定义事件：
  自定义事件名称 自定义json 创建 自定义事件
  可以点击解绑按钮，解绑事件
APP设置：
  修改APP 名称 崩溃 http 还有自定义事件的开启和关闭
